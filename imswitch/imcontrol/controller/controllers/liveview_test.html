API Documentation

Start Live Stream

POST /liveview/startLiveView
{
  "detectorName": "Camera",  // optional, uses first detector if not specified
  "protocol": "binary",      // binary, jpeg, mjpeg, webrtc
  "params": {                // optional protocol-specific params
    "compression_algorithm": "lz4",
    "compression_level": 0,
    "subsampling_factor": 4,
    "throttle_ms": 50,
    "jpeg_quality": 80
  }
}

Stop Live Stream

POST /liveview/stopLiveView
{
  "detectorName": "Camera",
  "protocol": "binary"
}

Get Stream Parameters

GET /liveview/getStreamParams
GET /liveview/getStreamParams?protocol=binary

Set Stream Parameters

POST /liveview/setStreamParams
{
  "protocol": "binary",
  "params": {
    "compression_algorithm": "lz4",
    "subsampling_factor": 4,
    "throttle_ms": 50
  }
}

Get Active Streams

GET /liveview/getActiveStreams

MJPEG Video Feed (HTTP)

GET /liveview/video_feeder?startStream=true
GET /liveview/video_feeder?startStream=true&detectorName=Camera

Legacy API (Backward Compatible)

// Settings Controller endpoints still work
POST /settings/setStreamParams
GET /settings/getStreamParams
GET /recording/video_feeder  // Delegates to LiveViewController
